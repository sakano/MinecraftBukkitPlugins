# MinecraftBukkitPlugins
MinecraftのBukkitサーバー向けプラグインです。
CB 1.7.10-R0.1, jdk 1.8.0_25の環境でテストしています。
サーバのjavaのバージョンが低かったりすると動きません。

##UpdateInfo
ログイン時に更新情報を出す

##GlassBack
ガラスや板ガラスを破壊したときにシルクタッチなしでもドロップする

##MineAssistEx
鉱石を掘った時に、隣接する同種の鉱石が一括で採掘される

##tpPlayers
指定されたワールド(fromworlds)にログインした時に、指定のワールド(toWorld)のリスポン地点にテレポートさせる
